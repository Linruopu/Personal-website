## What is Agent Workflow?

When people talk about "AI agents" today, they're often referring to a spectrum of capabilities — from simple LLM wrappers that call a single tool, all the way to autonomous systems that plan, execute, and self-correct over dozens of steps.

Agent Workflow sits at the structured end of this spectrum. It's the engineering discipline of designing **multi-step, decision-aware execution graphs** where each node is a specific capability (an LLM call, a tool invocation, a data fetch, a human approval step) and edges represent the flow of context and control.
